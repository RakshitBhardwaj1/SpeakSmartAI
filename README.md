# SpeakSmartAI
SpeakSmartAI is an AI-powered interview coaching system that evaluates a candidate’s interview performance by analyzing both speech delivery and body language.
Unlike traditional interview preparation tools, this system focuses on holistic development by assessing tone, word stress, intonation, emotions, posture, and gestures in real-time.

It helps users understand not just what they say, but how they say it and present themselves.

🚀 Key Features

🎙️ Speech Recognition – Converts spoken responses into text using ASR

📊 Word Stress & Intonation Analysis – Evaluates pronunciation, pitch, pauses, and fluency

😊 Tone & Emotion Detection – Detects confidence, nervousness, calmness, etc.

🧍 Body Language & Pose Detection – Analyzes posture, hand movement, head tilt, and eye contact

🧮 AI-Based Scoring System – Provides an overall interview performance score

📈 Feedback Dashboard – Suggests personalized improvement tips
